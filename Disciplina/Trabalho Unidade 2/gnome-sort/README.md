# Gnome Sort

A ideia vem da analogia de um duende de jardim (gnome) que organiza seus vasos de flores. 

A organizacao é feita pelos seguintes métodos: 

- Ele olha para o vaso ao lado e o anterior, se eles estão na ordem certa, ele dá um passo para a frente. Senão, ele troca os vasos e volta um passo. 

- Se ele está no começo, ele só anda para frente, se está no final, termina. 

Um exemplo usando o array = {5,3,2,4}
